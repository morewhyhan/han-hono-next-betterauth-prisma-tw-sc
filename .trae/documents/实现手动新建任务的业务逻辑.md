# 整合services到hooks中的实施计划

## 1. 创建功能对应的hooks

* 创建 `hooks/useTasks.ts`，整合任务相关的API调用和状态管理

* 创建 `hooks/usePersonality.ts`，整合个性设置相关的API调用和状态管理

## 2. 更新组件调用

* 更新 `app/(main)/home/page.tsx`，使用useTasks hook替代taskService

* 更新 `app/(main)/tasks/[id]/page.tsx`，使用useTasks hook替代taskService

* 更新 `app/(main)/settings/personality/page.tsx`，使用usePersonality hook替代personalityService

## 3. 删除services文件夹

* 删除 `services/taskService.ts`

* 删除 `services/personalityService.ts`

* 删除services文件夹

## 4. 验证构建

* 运行 `npm run build`，确保项目能正常构建

## 5. 验证功能

* 确保所有功能正常工作

## 预期结果

* 项目结构更简洁，移除了冗余的services层

* 所有API调用都通过hooks管理，符合React生态最佳实践
